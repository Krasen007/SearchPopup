## Summary (80 chars)

## Full Description
Revolutionize your web browsing experience with Search Popup! This intelligent Firefox add-on appears instantly when you select any text, providing smart contextual actions that save you time and boost your productivity.

**Blazing-Fast Performance**
- Zero-lag operation with advanced DOM caching
- Pre-compiled regex patterns for instant text parsing
- Memory-efficient design with automatic cleanup
- Optimized for Firefox's rendering engine

**Powerful Text Actions**
- **Multi-Engine Search**: Choose between Google, DuckDuckGo, Bing, Brave, Qwant, Ecosia, and more
- **Smart Copy**: One-click clipboard copy with intelligent fallback handling
- **URL Detection**: Automatically detects and highlights links in your selection
- **Unit Converter**: Convert weights, temperatures, speeds, distances, volumes, and more
- **Currency Exchange**: Live rates for 100+ fiat currencies
- **Crypto Support**: Real-time Bitcoin, Ethereum, and major cryptocurrency prices
- **Time Zone Tool**: Convert any time zone to your local time instantly

**Firefox-Optimized Design**
- Adaptive theming that perfectly matches any website (light/dark mode)
- Smart positioning with directional arrows that never obstruct content
- Smooth animations leveraging Firefox's CSS performance
- Responsive design that works on any screen size

**Privacy-First Approach**
- All processing happens locally on your device
- No data sent to external servers
- Minimal permissions (clipboard write, storage only)
- Secure external links with `noopener,noreferrer`
- Absolutely no tracking, analytics, or telemetry
